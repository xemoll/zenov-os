# ZenovOS 0.1.1 version scope

Version 0.1.1 is the correctness and integration release for the existing BIOS/i686 foundation. It closes memory-permission enforcement, userspace cleanup, foreground-process ABI, interrupted-write recovery and Zenov-source application execution.

Features requiring a scheduler, independent address spaces, new hardware stacks, a graphical subsystem, networking or an incompatible filesystem format are outside this version. They must begin in a later version with their own architecture and migration contracts.
